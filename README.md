# Gator CLI Tool

Gator is an RSS feed aggregator in Go. Gator? AggreGATOR.

## Features

- one
- two

### Prerequisites

You will need to have Go and PostgreSQL installed to use Gator.

### Installation

You can use go install to install the Gator CLI tool.

```
go install
```

## Usage

There are a handful of commands you can use in Gator.

```
gator register nathan
gator addfeed https://techcrunch.com/feed/
gator browse
```
